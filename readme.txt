The Shipping Method for WooCommerce UPS is a Wordpress Plugin that integrate the UPS service, it will calculate the shipping cost and the delivery time for your customer.

== Description ==
All of the domestic shipping options are supported: 3 Day Select, Ground, 2nd Day Air and Next Day Air. You can anytime, disable/enable one or all of these options.

== Installation ==

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `wc-ups-shipping-method.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `wc-ups-shipping-method.zip`
2. Extract the `wc-ups-shipping-method` directory to your computer
3. Upload the `wc-ups-shipping-method` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard

== Frequently Asked Questions ==
**How to get the UPS Access Key?**
1. Login to your UPS Account.
2. Go to My UPS -> Account Summary.
3. Under UPS Account Details, click on Add an Existing UPS Account.
4. Go to https://www.ups.com/upsdeveloperkit
5. Click on Request an access key.
6. The key will be sent to your account email.
